Adds a primary contact to an event in The Events Calendar by Modern Tribe.

== Description ==

Add name, phone and email for a primary contact to an event. Requires The Events Calendar by Modern Tribe.

== Installation ==

1. Upload `tlt-emp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
